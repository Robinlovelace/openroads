# OpenRoads Data Processing

This repository contains scripts to process OS OpenRoads data.

Data source: [OS OpenRoads](https://osdatahub.os.uk/data/downloads/open/OpenRoads)

## Scripts

- `get_openroads.R`: Downloads and subsets OpenRoads data for a specific area (e.g., Leeds).
- `split_openroads_by_authority.R`: Splits the national OpenRoads dataset into separate GeoPackages for each transport authority.
